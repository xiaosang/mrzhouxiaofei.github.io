HTTP 协议概述

HTTP 超文本传输协议，学过计算机网络的应该知道，所谓协议，就是指通信双方共同遵循的规范。HTTP 协议就是浏览器和服务器之间进行 “沟通” 的一种规范，它是互联网的基础协议，学习 Web 开发的都应当熟悉 HTTP 协议。
